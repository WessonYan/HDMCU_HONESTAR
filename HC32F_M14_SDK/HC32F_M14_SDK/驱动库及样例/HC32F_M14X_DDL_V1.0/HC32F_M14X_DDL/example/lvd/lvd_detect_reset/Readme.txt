功能描述
================================================================================
本样例主要展示MCU的低电压检测（LVD）的低电压复位功能

说明：
开启低电压检测
调整VCC输入电压，触发LVD的复位

================================================================================
测试环境
================================================================================
测试用板:
---------------------
TEST_M002_64L_V10

辅助工具:
---------------------
直流电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  示波器连接P01(CN1 3),稳压直流电压设置5V，连接测试版VCC
2）  打开工程编译，并将代码下载到目标芯片中，使全速运行。
3）  缓慢调整电源从5V到2.8V，示波器观察测试GPIO输出波形，连接LVD复位情况。
4）  缓慢调整电源从2.8V到5V，示波器观察测试GPIO输出波形，连接LVD复位情况。
