功能描述
================================================================================
本样例主要展示MCU的低电压检测（LVD）的低电压中断功能

说明：
开启低电压检测，两个通道同时使能，设置不同的电压阈值
--------------------------------
通道  下降电压阈值  上升电压阈值
LVD1      3.0V          4.2V
LVD2      3.0V          4.7V
--------------------------------
调整VCC输入电压，触发LVD的中断


================================================================================
测试环境
================================================================================
测试用板:
---------------------
TEST_M002_64L_V10

辅助工具:
---------------------
直流电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  打开工程编译下载代码至测试版中
2）  示波器连接测试GPIO P00(CN1 2)和P01(CN1 3)
3）  稳压流电压输出5V接入测试版VCC
4）  缓慢调低电源电压5V至2.8V，示波器观察测试GPIO了解进入LVD中断情况
5）  缓慢升高电源电压2.8V至5V，示波器观察测试GPIO了解进入LVD中断情况
