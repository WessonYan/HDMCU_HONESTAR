功能描述
================================================================================
CT定时器 RT io mode 6模式
说明：
本样例主要展示通过CT-RT i0 mode 6模式输出波形的使用流程。
通道2通过软件触发开始计数。
通道2输出下降沿，通道3开始计数；通道2输出上升沿，通道3停止计数。
