功能描述
================================================================================
CT定时器 PWM模式
说明：
本样例主要展示通过CT-PWM iomode 1模式输出波形的使用流程。
RT通道2输出时钟作为 PWM通道0的外部输入时钟。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
TEST_M002_64L_V10
说明：外部接4M晶振

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1) 连接TIOA2_0(CN2 4)和 TIOB0_0(CN2 8),TIOA1_0(CN2 6)连接P00(CN1 2)
   示波器连接TIOA0_0(CN2 9),观察输出波形
2) Rebuild工程，下载运行程序
3）TIOA1_0(CN2 6)拉低，输入一个下降沿（由代码GPIO输入）
4）观察TIOA0_0(CN2 9)输出的PWM波形


================================================================================
注意
================================================================================
注意连线，如果连线错误，则PWM功能无法正常运行
