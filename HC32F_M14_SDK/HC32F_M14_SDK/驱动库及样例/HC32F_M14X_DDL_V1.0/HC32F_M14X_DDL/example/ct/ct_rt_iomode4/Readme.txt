功能描述
================================================================================
CT定时器 RT io mode 4模式
说明：
本样例主要展示通过CT-RT i0 mode 4模式输出波形的使用流程。
通道2的输出作为通道3的trigger信号，下降沿通道3开始计数功能，上升沿通道3停止计数功能

使用步骤
================================================================================
1) 示波器连接TIOA2_0(CN2 4)和TIOA3_1(CN1 20),
   TIOB2_0(CN2 3)连接P00(CN1 2)
2) Rebuild工程，下载运行程序
4) 观察TIOA2_0(CN2 4)和TIOA3_1(CN1 20)输出的RT波形
