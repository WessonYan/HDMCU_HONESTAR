功能描述
================================================================================
CT定时器 PPG模式 io mode 2
说明：
本样例主要展示通过CT-PPG模式 io mode 2输出波形的使用流程。
通过TIOB0_0触发开始计数。

使用步骤
================================================================================
1) 示波器连接TIOA0_0(CN2 9)和TIOA1_0(CN2 6),
   连接TIOB0_0(CN2 8)和P00(CN1 2)
2）Rebuild工程，下载运行程序
3）观察TIOA0_0(CN2 9)和TIOA1_0(CN2 6)输出的波形
