功能描述
================================================================================
CT定时器 RT io mode 7模式
说明：
本样例主要展示通过CT-RT i0 mode 7模式输出波形的使用流程。
通道2通过TIOB2_0下降沿触发。
通道3通过通道2输出的上升沿触发开。

3) 观察TIOA2_0(CN2 4)和TIOA3_1(CN1 20)输出的RT波形
