功能描述
================================================================================
CT定时器 PPG模式 io mode 8
说明：
本样例主要展示通过CT-PPG模式 io mode 8输出波形的使用流程。
同时使用了CT channel 0的IO mode 2 作为输入控制计数信号，下降沿开始计数，上升沿
停止计数。
