功能描述
================================================================================
CT定时器 PWM模式
说明：
本样例主要展示通过CT-PWM模式输出波形的使用流程。
本样例采用P00产生的下降沿触发计数功能。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）示波器连接TIOA1_1(CN3 A3)，TIOB1_1(CN3 A2)连接P00(CN6 10)接收外部触发信号
1) Rebuild工程，下载程序，
2）代码main函数内“ m_u32CntIntTrg = 0u;”处放置断点
3）运行至断点处，观察中断使能情况下TIOA1_1输出的PWM波形
4) 通过IDE查看m_u32CntIntUnder、m_u32CntIntDuty、m_u32CntIntTrg计数
5) 运行至程序结束，观察中断不使能时TIOA1_1输出的PWM波形
