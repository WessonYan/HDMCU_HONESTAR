CT定时器 RT式
说明：
本样例主要展示通过CT-RT 32bit 模式输出波形的使用流程。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1) Rebuild工程，下载程序，示波器连接TIOA0_0（CN3 A0）
2）代码231行“ m_u32CntIntTrg = 0;”处放置断点
3）运行至断点处，观察中断使能情况下TIOA0_0输出的RT波形
4) 通过IDE查看m_u32CntIntUnder、m_u32CntIntTrg计数
5) 运行至程序结束，观察中断不使能时TIOA0_0输出的RT波形
