功能描述
================================================================================
本样例主要展示UID的获取功能。
说明：
   读取UID.

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）在"while(1)"行设置断点；
4）运行；
5）使用IDE中变量查看功能观察变量[au32Uid]的值。

================================================================================
注意
================================================================================
获取的UID值为芯片的80 bits编号（十六进制）。
