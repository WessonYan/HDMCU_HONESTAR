功能描述
================================================================================
本样例主要展示外设（本例中使用GPIO外设）时钟的停止与开启功能及流程。
说明：
   1、当外设的时钟停止时,对外设的寄存器进行读或写操作将失败（CPU读到的数据不是该寄存器的真实值，同时写操作也被禁止）。
   2、当外设时钟开启时，对外设寄存器的操作有效。

================================================================================
使用步骤
================================================================================
1）连接测试pin到示波器；
2）打开工程并重新编译；
2）启动IDE的下载和调试功能并运行；
3）程序会进入最后一行“while(1)”死循环；
4）观察示波器波形变化。
