功能描述
================================================================================
本样例主要展示中断优先级功能。
说明：
此样例展示芯片涉及的以下5个中断优先级中，低优先级中断被高优先级中断打断的过程。
---------------------------
NMIX      系统请求       高
INT00_2   中断优先级0     
INT08_2   中断优先级1
SWWDT     中断优先级2
DT        中断优先级3    低
---------------------------

================================================================================
使用步骤
================================================================================
1）打开样例工程，检查并修改interrupts_hc32f15.h,使用到的优先级按以下定义
    #define DDL_IRQ_LEVEL_DTIM            3u
    #define DDL_IRQ_LEVEL_SWDT            2u
    #define DDL_IRQ_LEVEL_EXT815          1u
    #define DDL_IRQ_LEVEL_EXT07           0u

2）INT00_2(CN6 10)，INT08_2(CN6 2)和NMIX(CN6 8) 上拉
3）打开main.c分别在NMIX，Exint通道0，Exint通道8，SWDT，DT中断服务程序内放置断点
4）编译，下载运行代码
5）运行代码后，首先进入DT中断服务程序，继续运行代码
6）然后进入SWDT中断服务程序，继续运行代码
7）拉低引脚 INT08_2(CN6 2)后，进入外部中断通道8的中断服务程序，继续运行代码
8）拉低引脚 INT00_2(CN6 10)后，进入外部中断通道0的中断服务程序，继续运行代码
9）拉低引脚NMIX后，进入NMIX中断服务程序
10）结束
