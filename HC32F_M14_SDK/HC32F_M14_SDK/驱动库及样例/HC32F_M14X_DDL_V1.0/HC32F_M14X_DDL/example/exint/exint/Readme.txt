功能描述
================================================================================
本样例主要展示外部下降沿触发的外部中断功能。



================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）连接INT01_2(CN6 9) 和 P00(CN6 10)
2）打开工程并重新编译，在函数Main_ExtInt0Callback(void)内放置断点
3）启动IDE的下载和调试功能，全速运行；
4）观察代码运行后是否会因为P00产生的下降沿进入exint通道1的中断回调函数
