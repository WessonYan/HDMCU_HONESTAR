功能描述
================================================================================
本样例主要展示DMAC硬件请求传输操作。
说明：
本样例主要展示如何使用MSC触发DMAC硬件请求传输操作。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）运行；
4）程序运行至“while (trans_done != enDmaTransStat);”下一句，表面DMAC传输完成；
5）比较源数组与目的数组里的数据是否一致。

================================================================================
注意
================================================================================
本demo需连接MSC通道5的SOT和SIN引脚
