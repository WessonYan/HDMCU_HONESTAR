本样例主要展示MCU的Fast GPIO的输出功能。
说明：

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  连接测试PIN P60(CN3 A4)到示波器,观察fast gpio输出
2）  打开工程，设置工程优化等级为“high”优化类型为“speed”
3）  编译下载运行代码
4）  通过示波器观察测试PIN输出的波形
