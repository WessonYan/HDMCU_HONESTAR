I2C从机查询模式数据接收发送样例
说明：
本样例主要展示I2C从机查询模式的数据接收和发送
此样例需要i2c_master_polling配合演示

================================================================================
测试环境
================================================================================
测试用板:
---------------------
TEST_M002_64L_V10

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1) 硬件连接

    主机端(运行样例"i2c_master_polling")   从机端(运行样例"i2c_slave_polling")
    SCK5_0(CN1 10)   -------------------   SCK5_0(CN1 10)
    SOT5_1(CN2 21)   -------------------   SOT5_1(CN2 21)

2）从极端运行样例"i2c_slave_polling"
3）主机端运行样例"i2c_master_polling"
4）主机端在样例"i2c_master_polling"第380行和第383行放置断点，观察代码停止位置，
   得出主机端和从机端的数据发送接收事务是否正确完成。

================================================================================
1）SCK和SOT需要接上拉电阻
