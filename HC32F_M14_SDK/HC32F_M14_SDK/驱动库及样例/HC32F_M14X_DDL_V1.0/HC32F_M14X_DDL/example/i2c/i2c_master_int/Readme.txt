功能描述
================================================================================
I2C主机中断模式数据发送接收样例
说明：
本样例主要展示I2C主机中断模式的数据发送和接收
此样例需要i2c_slave_polling或者i2c_slave_ini配合演示

================================================================================
测试环境
================================================================================
测试用板:
---------------------
TEST_M002_64L_V10


辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1) 硬件连接

    主机端(运行样例"i2c_master_ini")       从机端(运行样例"i2c_slave_ini")
    SCK5_0(CN1 10)   -------------------   SCK5_0(CN1 10)
    SOT5_1(CN2 21)   -------------------   SOT5_1(CN2 21)

2）从极端运行样例"i2c_slave_ini"
3）主机端运行样例"i2c_master_ini"
4）主机端在样例"i2c_master_ini"第528行和第531行放置断点，观察代码停止位置，
   得出主机端和从机端的数据发送接收事务是否正确完成。

================================================================================
