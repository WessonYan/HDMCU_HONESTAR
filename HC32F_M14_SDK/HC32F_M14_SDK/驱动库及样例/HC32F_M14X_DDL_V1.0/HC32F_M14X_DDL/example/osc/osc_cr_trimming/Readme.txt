功能描述
================================================================================
本样例主要展示内部高速CR频率Trimming操作。
说明：
本样例主要展示如果使用外部OSC对内部HiCR进行频率Trimming的操作流程。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
示波器（如：Tektronix DPO2024B）

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）运行；
4）程序运行至“while(1);”无限循环；
5）使用示波器测量P73（第60管脚），观察频率是否为4MHz（+-2%）。

================================================================================
注意
================================================================================
如使用非4MHz外部OSC作为主时钟，请自行配置PLL倍频数，但请保证芯片最大工作频率小于等于48MHz
