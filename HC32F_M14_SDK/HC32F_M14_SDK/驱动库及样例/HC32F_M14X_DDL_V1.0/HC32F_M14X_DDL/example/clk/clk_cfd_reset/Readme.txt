功能描述
================================================================================
本样例主要展示时钟失效检测复位功能。

说明：
时钟失效检测功能检测时钟为外部高速时钟和外部低速时钟。在一定的时间内，如果被检测
时钟没有上升沿，则认为该时钟已经失效，此时产生系统复位。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  编译、下载并运行；
2）  观察60秒左右，LED灯一直处于熄灭状态；
3）  短接XHO与XHI引脚；
4）  LED3闪烁；
5）  不短接XHO与XHI, 复位测试板；
6）  观察60秒，LED3一直处于熄灭状态；
7）  短接XLO与XLI引脚；
8）  LED3闪烁；

================================================================================
注意
================================================================================
为了保证样例功能正常：
1）若更换测试板，请根据LED对应的引脚，修改如下LED灯相关宏定义；
   #define Led_Init() Gpio_InitIO(2, 4, GpioDirOut, FALSE, FALSE) 
   #define Led_On()   Gpio_SetIO(2, 4, TRUE)
   #define Led_Off()  Gpio_SetIO(2, 4, FALSE)
