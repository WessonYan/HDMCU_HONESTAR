功能描述
================================================================================
本样例主要展示不同主时钟时钟切换。

说明：
无

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
示波器Tektronix MDO3054

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  测试板P00引脚与示波器相连，设置信号触发;
2）  编译、下载;
3）  155行处设置断点，并运行；
4）  示波器显示波形脉宽为对应主时钟周期；
5）  继续执行步骤3）和 4）

================================================================================
注意
================================================================================
为了保证样例功能正常：
1）设置编译器优化级别为最高；
2）若更换测试板，请根据对应的测试引脚，修改如下相关宏定义；
   #define FAST_IO_INIT( )   FGpio_InitIO(0, 0, GpioDirOut)
   #define FAST_IO_PORT_PIN  bM0P_FASTIO_FSET0_P00
