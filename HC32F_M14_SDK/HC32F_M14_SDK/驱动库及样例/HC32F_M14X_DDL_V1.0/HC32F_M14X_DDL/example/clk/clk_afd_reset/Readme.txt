功能描述
================================================================================
本样例主要展示时钟频率异常检测复位功能。

说明：
时钟频率异常检测功能以外部高速时钟为计数时钟，检测时钟外部高速时钟。如果计数值超
出设定的范围，则认为被检测时钟频率异常，此时产生系统复位或中断。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）编译、下载并运行；
2）LED3熄灭；
3）取消如下代码注释；
   Clk_SetAfdWindow(1024 - 200, 1024 - 100); /* abnormal window */
   或者
   Clk_SetAfdWindow(1024 + 100, 1024 + 200); /* abnormal window */
4）重新编译、下载并运行；
5）LED3闪烁。

================================================================================
注意
================================================================================
为了保证样例功能正常：
1）本样例程序运行的结果和MCU的Clock设置相关，测试板MCU内部高速时钟频率已调校，外
   部晶振为4MHz频率；
2）若更换测试板，请根据LED对应的引脚，修改如下LED灯相关宏定义；
   #define Led_Init() Gpio_InitIO(2, 4, GpioDirOut, FALSE, FALSE) 
   #define Led_On()   Gpio_SetIO(2, 4, TRUE)
   #define Led_Off()  Gpio_SetIO(2, 4, FALSE)
