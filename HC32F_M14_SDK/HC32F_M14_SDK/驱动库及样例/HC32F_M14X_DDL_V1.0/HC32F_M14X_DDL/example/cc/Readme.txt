功能描述
================================================================================
本样例主要展示MCU的计时计数器（CC）的计时功能，使用中断模式完成。每次
计数完成后，翻转测试IO输出。

说明：

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  示波器1路连接测试PIN 
2）  打开工程编译并运行
3）  示波器观察输出频率，计算是否符合理论值。
