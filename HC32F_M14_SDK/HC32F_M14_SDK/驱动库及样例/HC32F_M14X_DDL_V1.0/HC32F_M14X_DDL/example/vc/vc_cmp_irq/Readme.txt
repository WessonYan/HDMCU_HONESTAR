功能描述
================================================================================
本样例主要展示MCU的电压比较（VC）功能。 比较两个电压的大小，通过中断方式来输出
结果。

说明：
本样例负端电压选择AVCC/4

================================================================================
测试环境
================================================================================
测试用板:
---------------------
TEST_M002_64L_V10

辅助工具:
---------------------
示波器
直流稳压电源

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  示波器A路连接CAOUT(CN1 10), 观察硬件比较结果；
     示波器B路连接GPIO P01(CN1 3)观察进入VC中断情况。
2)   直流稳压电源输出正端连接VC_INP(CN1 11), 负极连接GND，直流稳压电源输出0.5V。
3）  打开工程编译并运行
4）  观察示波器A路，为低电平。
     调整电源输入电压到大于（AVCC/4）1.3V，CAOUT(CN1 10)变高，测试GPIOP01(CN1 3)
     也输出高，说明检测到输出上升沿进入中断；
5）  调整电源输入电压到小于（AVCC/4）1.3V，CAOUT(CN1 10)变低，测试GPIOP01(CN1 3)
     也输出低，说明检测到输出下降沿进入中断；
