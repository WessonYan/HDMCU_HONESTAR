功能描述
================================================================================
本样例主要展示低功耗IDLE模式。
说明：
设置LED灯管脚P24 为低 打开LED灯
并使用P00管脚控制MCU进入低功耗(idle)状态（高电平不进入，低电平进入）
通过UART接口往板子上的UART5发送数据唤醒MCU后设置P24为高关闭LED灯，标识唤醒成功。

4) 5S后将P00拉高，代表芯片被外部中断唤醒，此时led灯亮
