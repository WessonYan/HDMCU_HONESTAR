功能描述
================================================================================
本样例主要低功耗进入以及低功耗唤醒功能
说明：
进入低功耗模式，并通过外部中断唤醒

================================================================================
测试环境
================================================================================
测试用板:
---------------------
TEST_M002_64L_V10

辅助工具:
---------------------
示波器

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）INT00_1引脚上拉，TOGGLE GPIO P51通过示波器查看芯片工作状态；
3）下载代码并使代码全速运行；
4）通过示波器查看芯片工作状态，GPIO P51输出低后，芯片进入STANDTIMER模式
5）使INT00_1引脚下拉，此时GPIO P51输出高，芯片被唤醒
6）分别测试HICR,HSXT,LICR,LSXT,PLL。
================================================================================
注意
================================================================================
1.STOP模式所有时钟会关闭，唤醒需要等待时钟源稳定，为保证其他没使用时钟关闭，测试时重新上电芯片
