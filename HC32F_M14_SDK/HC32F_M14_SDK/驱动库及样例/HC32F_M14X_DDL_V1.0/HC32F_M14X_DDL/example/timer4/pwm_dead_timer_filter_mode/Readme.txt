功能描述
================================================================================
Timer4定时器 脉冲宽度调制模块PWM

说明：
样例配置比较输出模块OCO奇偶通道，并通过PWM死区定时器滤波模式，输出其波形。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
示波器Tektronix MDO3054

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）测试板CN5-PWM0、CN-PWM1引脚与示波器相连；
2）编译、下载并运行；
3）观察示波器，PWM0、PWM1周期相同，占空比为1/2，极性相反，且周期时间为CNT周期2倍。
