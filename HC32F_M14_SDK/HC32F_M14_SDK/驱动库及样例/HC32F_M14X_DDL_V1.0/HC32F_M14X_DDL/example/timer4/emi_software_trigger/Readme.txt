功能描述
================================================================================
Timer4定时器 电机控制紧急输入EMI软件触发功能。

说明：
样例配置比较输出模块OCO奇数通道1和PMW1输出波形，然后通过软件触发，产生EMIF中断。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
示波器Tektronix MDO3054

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）测试板CN5-PWM1引脚与示波器相连；
2）编译、下载并运行；
3）观察示波器，示波器显示PWM脉冲。
4）按下USER按键并释放。
5）观察示波器，示波器显示低电平信号。
6）按下USER按键并释放。
7）观察示波器，示波器显示PWM脉冲。
8）重复4）~5）
