功能描述
================================================================================
Timer4定时器 脉冲宽度调制模块PWM

说明：
PWM设置为直通模式和死区定时器模式时，PFSR可用于16位重载计数器功能。
样例配置PWM为死区定时器模式，设置PFSR，使其产生定时器中断。

================================================================================

---------------------

---------------------

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）测试板CN5-PWM0引脚与示波器相连；
2）编译、下载并运行；
3）观察示波器，PWM0输出周期脉冲，且周期时间为重载定时器周期一致。
