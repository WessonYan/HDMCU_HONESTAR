功能描述
================================================================================
Timer4定时器 ADC启动触发单元ADCT -延迟模式

说明：
ADC启动触发单元ADCT-延迟模式：当计数器计数值与所选的输出比较值寄存器OCCR相等时, 
加载ATVR值到启动延迟计时器并开始计时，当ATVR所指定的计时完成时，输出ADC转换启动
信号。

================================================================================
使用步骤
================================================================================
1）测试板CN3-A1引脚输入小于3.3V信号；
2）编译、下载并运行；
3）通过IDE中的变量窗口观察u16AdcPriValue、m_u16AdcPriCnt和m_u16AdcPriErr，
   m_u16AdcPriErr值为0（表示ADC采样所有数据都为有效）、
   u16AdcPriValue等于对应ADC输入信号采样值（表示ADC采样数据与输入一致性）、
   m_u16AdcPriCnt值一直递增（表示Timer4 ADCT一直触发ADC采样）。
