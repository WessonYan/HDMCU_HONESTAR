功能描述
================================================================================
Timer4定时器 ADC启动触发单元ADCT-比较模式

说明：
ADC启动触发单元ADCT-比较模式：计数器单元CNT与ADCT比较寄存器值匹配时，产生ADC转换
启动信号。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）测试板CN3-A0引脚输入小于3.3V信号；
2）编译、下载并运行；
3）通过IDE中的变量窗口观察u16AdcScanValue、m_u16AdcScanCnt和m_u16AdcScanErr，
   m_u16AdcScanErr值为0（表示ADC采样所有数据都为有效）、
   u16AdcScanValue等于对应ADC输入信号采样值（表示ADC采样数据与输入一致性）、
   m_u16AdcScanCnt值一直递增（表示Timer4 ADCT一直触发ADC采样）。

注意
================================================================================
