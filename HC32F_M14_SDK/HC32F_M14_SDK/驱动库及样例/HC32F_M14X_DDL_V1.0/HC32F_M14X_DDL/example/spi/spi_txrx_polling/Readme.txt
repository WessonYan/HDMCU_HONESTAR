功能描述
================================================================================
本样例主要展示MCU的SPI 数据的发送接收功能，采用轮询模式。

说明：
	1、本样例使用通道4（主）发送数据，通道5（从）接收数据
	2、本样例发送接收采用8数据位，位方向MSB，波特率200K，SCK检测电平H，等待时间3
	3、本样例采用轮询模式，发送一个数据后，马上接收一个数据，并存到接收buffer

================================================================================
使用步骤
================================================================================
1）  连接SCK4_1 和 SCK5_0, SOT4_1和SIN5_0。
2）  打开工程编译并运行
3）  当程序运行几秒后，中断程序。程序停留在最后的while（1）。观察u8RxBuff 是否和
     u8TxBuff 数据相同。
