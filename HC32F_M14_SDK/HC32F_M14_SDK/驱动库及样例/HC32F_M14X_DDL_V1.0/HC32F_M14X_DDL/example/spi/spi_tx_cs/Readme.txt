功能描述
================================================================================
本样例主要展示MCU的SPI 片选（SPI chip selection）功能。

说明：
	1、本样例使用通道4（主）来展示spi片选功能
	2、片选有效保持0，片选引脚无效状态L，建立时间3，保持时间3，最小释放时间5
	每发送3个byte，切换cs pin
	3、通过示波器观察SPI数据线和CS pin的关系

================================================================================
测试环境
================================================================================
测试用板:
---------------------
（TEST-M002-64L V10）

辅助工具:
---------------------
Tektronix DPO 2024B

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）  连接SCS40_1,SCS41_1, SCK4_1,SOT4_1 到示波器。
2）  打开工程编译并运行
3）  通过示波器观察各个PIN输出的波形。（每发送3个byte 数据，会交换切换一个CS PIN
     （数据有效时电平低））
4）  修改建立时间，保持时间，取消选定时间，每通道发送字节数，观察波形。
