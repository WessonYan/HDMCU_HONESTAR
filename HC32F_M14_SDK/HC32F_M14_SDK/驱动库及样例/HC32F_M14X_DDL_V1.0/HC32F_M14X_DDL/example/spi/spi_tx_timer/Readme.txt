功能描述
================================================================================
本样例主要展示MCU的SPI 数据的定时发送功能。

说明：
	1、本样例使用通道4（主）来进行定时发送功能
	2、本样例发送接收采用8数据位，位方向MSB，波特率100K，SCK检测电平H，等待时间3
	3、本样例定时发送的时钟分频比为16分之一PCLK，定时时间100，发送三字节

================================================================================
使用步骤
================================================================================
1）  连接SCK4_1， SOT4_1, 到示波器。
2）  打开工程编译并运行
3）  观察SPI数据是否周期性发送。
4）  修改count，观察周期是否变化。
