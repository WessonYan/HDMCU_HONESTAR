功能描述
================================================================================
本样例主要展示MCU的SPI 数据的发送接收功能，采用中断模式。

说明：
	1、本样例使用通道6（主）发送数据并使能发送中断，通道5（从）接收数据并使能接
	收中断
	2、本样例发送接收采用8数据位，位方向MSB，波特率100K，SCK检测电平H，等待时间3
	3、本样例采用中断模式，通道6的发送中断每次发送一个数据后除能本通道发送中断，
	通道5的接收中断接收数据后存到接收buffer，并使能通道6的发送中断

================================================================================
使用步骤
================================================================================
1）  连接SCK6_1 和 SCK5_0, SOT6_1和SIN5_0。
2）  打开工程编译并运行
3）  当程序运行几秒后，中断程序。程序停留在最后的while（1）。观察u8RxBuff 是否和
     u8TxBuff 数据相同。
