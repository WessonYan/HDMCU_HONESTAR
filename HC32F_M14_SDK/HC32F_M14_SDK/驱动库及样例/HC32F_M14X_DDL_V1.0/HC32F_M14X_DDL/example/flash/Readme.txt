功能描述
================================================================================
本样例主要展示FLASH操作。
说明：
本样例主要展示对FLASH进行页擦除、写和读操作流程。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）运行；
4）程序运行至“while(1);”无限循环，（表明写、读正确）。

================================================================================
注意
================================================================================
在使用过程中，亦可以在程序每一步操作后设置断点，通过IDE中的Memory窗口观察FLASH中的数据的擦除、读和写是否正确。
