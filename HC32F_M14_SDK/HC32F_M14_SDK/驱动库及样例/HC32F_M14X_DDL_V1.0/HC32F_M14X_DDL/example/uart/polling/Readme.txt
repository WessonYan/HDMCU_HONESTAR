功能描述
================================================================================
本样例主要展示UART的轮询收发功能。
说明：
使用UART通道5和6进行收发

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
两根线按如下连接：
P05 SOT6_1 <--> P65 SIN5_1
P64 SOT5_1 <--> P06 SIN6_1


辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）在189行设置断点；
4）运行；
5）程序最终会在189行停止；
6）观察u8RxData中的值是否和u8TxData中一致。
