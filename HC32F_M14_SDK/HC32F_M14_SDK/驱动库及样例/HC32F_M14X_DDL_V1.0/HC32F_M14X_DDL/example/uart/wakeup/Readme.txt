功能描述
================================================================================
本样例主要展示UART的低功耗唤醒功能。
说明：
设置LED灯管脚P24 为低 打开LED灯
并使用P00管脚控制MCU进入低功耗状态（高电平不进入，低电平进入）
通过UART接口往板子上的UART5发送数据唤醒MCU后设置P24为高关闭LED灯，标识唤醒成功。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
唤醒UART通道：UART5
进入低功耗模式控制管脚：P00
唤醒测试管脚：P24


辅助软件:
---------------------
串口调试软件（比如CommPort）

================================================================================
使用步骤
================================================================================
1）将USB线和开发板相连；
2）打开工程并重新编译；
3）将代码下载到板子上，上电；
4）观察led在进入低功耗模式之前LED是亮的状态。
5）连接P00和VSS；
6）观察LED是亮的状态为（进入低功耗模式）；
7）打开PC端串口调试工具，设置端口（9600，无奇偶，8位,1 stop bit），发送数据；
8）观察LED灯熄灭（恢复正常模式）
