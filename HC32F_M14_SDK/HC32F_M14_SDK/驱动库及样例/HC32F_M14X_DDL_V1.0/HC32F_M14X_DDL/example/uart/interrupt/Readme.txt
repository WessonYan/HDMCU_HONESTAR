功能描述
================================================================================
本样例主要展示UART的中断收发功能。
说明：
使用UART通道5进行收发

================================================================================
测试环境
================================================================================
测试用板:
---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
两根线按如下连接：
P12 SIN5_0 <--> 电脑串口发送
P11 SOT5_0 <--> 电脑串口接收


辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）打开串口调试助手，发送任意9个字节数据；
4）观察串口接收串口是否收到“HUADA MCU!”字样；
