功能描述
================================================================================
本样例主要展示ADC优先级转换和扫描转换配置，ADC通用初始化配置，ADC 扫描转换重复采样，优先级转换多次采样示例。
说明：
    1、本样列选用外部A10(P70口)作为ADC优先级转换采样口，A8(P60口)作为ADC扫描转换采样口
    2、本样列使用内部avcc 5.0V，12位采样数据左对齐，采样频率选择PCLK0 16分频，优先级转换采样周期12个ADC时钟，扫描转换采样周期6个ADC时钟，
   优先级转换和扫描转换均选用FIFO级数1
    3、本样列进行了FIFO数据的读取，获取了FIFO数据有效性，通道，触发源，转换结果
   4、本样例优先级转换和扫描转换均为软件触发方式

================================================================================
使用步骤
================================================================================
1）正确接线；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能；
4）在__no_operation()处打上断点；
5）在Debug模式下，全速运行，能响应断点即说明在扫描转换重复采样模式下，优先级软件触发可中断扫描转换；
5）退出ADC Debug模式，结束；
