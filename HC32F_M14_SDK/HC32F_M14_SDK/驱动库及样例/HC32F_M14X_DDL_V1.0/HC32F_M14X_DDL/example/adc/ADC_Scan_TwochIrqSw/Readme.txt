功能描述
================================================================================
本样例主要展示ADC扫描转换配置，中断配置，ADC通用初始化配置，ADC 单次采样示例。
说明：
    1、本样列选用外部A10(P70口)和A8(P60口)作为ADC扫描采样口，采样通道模式为顺序采样
    2、本样列使用内部avcc 5.0V，12位采样数据左对齐，采样频率选择PCLK0 8分频，采样周期6个ADC时钟，
   chn 10采样时间寄存器为reg1，chn 8采样时间默认为reg0，选用FIFO级数2，连续采样
    3、本样列使能了扫描转换完成中断，进行了FIFO数据的读取，获取了FIFO数据有效性，通道，触发源，转换结果

测试环境
================================================================================

---------------------
SK-HC32F156-64L v10

辅助工具:
---------------------
Tektronix PWS2721

辅助软件:
---------------------
无

PIN脚连接:
---------------------
各通道P70脚连接外部输入V+，pin GND 接V-，P60脚连接外部输入V+。
================================================================================
使用步骤
================================================================================
1）正确接线；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能；
4）在Debug模式下，监视ADC采样结果，查看当前ADC量化结果，并确认是否先采样chn 8，再采样chn 10；
5）退出ADC Debug模式，结束；
