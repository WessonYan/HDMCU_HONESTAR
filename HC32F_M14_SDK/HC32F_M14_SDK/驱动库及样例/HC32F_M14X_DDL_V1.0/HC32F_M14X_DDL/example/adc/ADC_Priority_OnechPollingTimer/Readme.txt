功能描述
================================================================================
本样例主要展示ADC优先级转换timer触发配置，ADC通用初始化配置。
说明：
    1、本样列选用外部A10(P70口)作为ADC优先级转换采样口
    2、本样列使用内部avcc 5.0V，12位采样数据左对齐，采样频率选择PCLK0 8分频，优先级转换采样周期6个ADC时钟，选用FIFO级数1，使能timer触发，timer源为复合定时器0
    3、本样列进行了FIFO数据的读取，获取了FIFO数据有效性，通道，触发源，转换结果
   4、本样例配置了复合定时器0-重载定时器功能，单次模式

使用步骤
================================================================================
1）正确接线；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能；
4）在Debug模式下，监视ADC采样结果，查看当前ADC触发源；
5）退出ADC Debug模式，结束；
