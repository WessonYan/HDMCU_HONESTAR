功能描述
================================================================================
本样例主要展示ADC优先级转换外部触发配置，ADC通用初始化配置，ADC 重复采样示例。
说明：
    1、本样列选用外部A10(P70口)作为ADC优先级转换优先级2采样口，A8(P60口)作为优先级1采样口
    2、本样列使用内部avcc 5.0V，12位采样数据左对齐，采样频率选择PCLK0 8分频，优先级转换采样周期6个ADC时钟，选用FIFO级数2
    3、本样列进行了FIFO数据的读取，获取了FIFO数据有效性，通道，触发源，转换结果

---------------------

================================================================================
使用步骤
================================================================================
1）正确接线；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能；
4）在__no_operation()处打上断点；
5）在Debug模式下，全速运行，能响应断点即说明外部触发可打断软件触发；
6）退出ADC Debug模式，结束；
