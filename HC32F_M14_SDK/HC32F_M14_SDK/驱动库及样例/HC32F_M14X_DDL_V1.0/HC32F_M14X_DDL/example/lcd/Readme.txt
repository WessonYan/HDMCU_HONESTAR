功能描述
================================================================================
本样例主要展示LCD初始化配置，按内容显示、全屏显示或全屏清除等基本LCD DDRAM显示。
说明：
    1、本样例主要描述LCD按照结构体初始化的基本流程，及按内容显示、全屏显示或全屏
   清除等基本功能。
    2、本样例实现了4com、1/4占空比、1/3bise、TypeA波形的LCD液晶屏显示。
    3、驱动电路选择内部电阻。
    4、LCD时钟选择内部低速时钟。

================================================================================
使用步骤
================================================================================
1）选择正确的液晶屏（4com、1/4占空比、1/3bise、5v），按上图接线；
2）打开工程并重新编译；
3）启动IDE的下载和调试功能；
4）运行；
5）看到LCD液晶屏上像素按要求显示；
