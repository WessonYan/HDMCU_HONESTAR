功能描述
================================================================================
本样例主要演示STK硬件基本功能是否正常，包括：
1、XTH晶振；
2、XTL晶振；
3、USER按键；
4、RST按键；
5、LED（绿色）；
6、LCD显示；

================================================================================

HC32LFx7(9)x_STK
---------------------
（需要填）

---------------------

辅助软件:
---------------------
无

================================================================================
使用步骤
1、重新编译代码、下载运行
2、液晶全显,显示“HC32”字样,LED亮;
3、按下USER按键，LCD和LED均闪烁（1Hz）;
4、按下RST，恢复步骤2状态;
