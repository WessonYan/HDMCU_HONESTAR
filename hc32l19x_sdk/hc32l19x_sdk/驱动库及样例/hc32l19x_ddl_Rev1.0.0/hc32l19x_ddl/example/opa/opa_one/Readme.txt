功能描述
================================================================================
说明：
本样例主要展示OPA作为电压跟随器的功能

================================================================================
测试环境
================================================================================
测试用板:
---------------------
HC32LFx7(9)x_STK

辅助工具:
---------------------
示波器
信号发生器

辅助软件:
---------------------
无。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）运行;
4）通过示波器观察OPA_INP与OPA_OUT波形之间的关系

================================================================================
注意
端口定义：
OPA_INN——PC05
OPA_INP——PB00
OPA_OUT——PA04
接线：
OPA_INN与OPA_OUT短接之间短接不短接均可
信号源输入给OPA_INP,
通过示波器监测OPA_INP与OPA_OUT。
